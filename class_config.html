Detailed Description

Read and parse configuration file.

This class can be used to open and retrieve information from configuration file. The information in config file is usually string type, this class provides methods to convert these strings into corresponding inputs for program.

The structure of configuration file should be:

Keyword : Value(s)

int Config::open ( TString  filename)

Method for openning file.

Parameters
filenamename of input file
Returns
status of openning file

This method opens the configuration file; return 1 if there is any error during openning, 0 if everything ok.

It loops all lines in the file, finds the position of charater ':' to determine (keyword, value) pair, both in string type, and store in m_config member.

To comment a line in configuration file, use '#' at the beginning.
