4 A collection of tools for analyzing different types of outputs: MCNP tally output files, X-ray and gamma spectra,...
5 
6 ## Requirements
7 
8  * ROOT version 6.00 or later
9  * GCC version 4.7 or later (support C++11 standard)
10  * GNU make
11 
12 ## Install
13 
14 #### Linux
15  * Download and install ROOT6 from https://root.cern.ch/downloading-root .
16  * Checkout the source: `git clone https://github.com/phuong84/data-analysis`.
17  * Move to the `data-analysis` directory and type `make` to compile all, `make mcnp` to compile MCNPAnalysis or `make spec` to compile SpectrumAnalysis.
18  * The compiled binary files will be put to the `bin` directory.
19 
20 #### Windows
21  * Download and install Cygwin from https://cygwin.com/install.html
22  * Install GCC, Make, Git along with Cygwin.
23  * Download ROOT6 binary distribution `root_v6.00.02.win32gcc-gcc-4.8.tar.gz` from https://root.cern.ch/content/release-60002, and unzip the file.
24  * Open Cygwin terminal
25  * Open and edit the file `.bashrc` by adding these lines into it
26 ```bash
27  export ROOTSYS=/directory/of/ROOT/binary
28  export PATH=$PATH:$ROOTSYS/bin
29  export LD_LIBRARY_PATH=$ROOTSYS/lib
30 ```
31 (you may need to type `bash` after editing and closing the `~/.bashrc` file to update environment variables)
32  * Checkout the source: `git clone https://github.com/phuong84/data-analysis`.
33  * Move to the `data-analysis` directory and type `make` to compile all, `make mcnp` to compile MCNPAnalysis or `make spec` to compile SpectrumAnalysis.
34  * The compiled binary files will be put to the `bin` directory.
