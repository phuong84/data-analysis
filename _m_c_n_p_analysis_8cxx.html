void processMesh ( Config *  config)

This is the function for reading different tally mesh results, merging and plotting them. Configuration options:

  • File Name : name of MCNP mesh tally outputs (separate by ',')
  • Outputfile Name : name of histogram output file
  • Merging: do merge mesh tallies (true or false)
  • Make Ratio : create projection ratio plots between meshes
  • Make Plot : create plots for tally outputs (true or false)
  • Projection Plane : name of plane to make 2D projection plots on (e.g. "XY")
  • Projection Axis : name of axis to make 1D projection plots on (e.g. "X")
  • First Bin: the first bin to be included in projection
  • Last Bin: the last bin to be included in projection

void processTally ( Config *  config)

This is the function for reading MCNP tally output, with configuration options:

  • File Name : name of MCNP output file
  • Make Plot : create plots for tally output (true or false)
