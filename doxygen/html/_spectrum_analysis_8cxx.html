void processSpectrum ( Config *  config)

This is the function for reading and analysing spectra, with configuration options:

  • File Name : name of spectrum files
  • File Type : type of spectrum files
  • Peak Finding: find peaks
  • Peak Fitting: fit peak
  • Background Subtraction: subtract background
  • Efficiency Fitting: fit efficiency curve
  • Make Plots: make plots

Options for finding peaks:

  • Sigma : sigma of peak finding method
  • Threshold : threshold of peak finding method

Options for fitting peaks:

  • Minimum Range : minimum value of range
  • Maximum Range : maximum value of range
  • Fitting Formula : fitting formula
  • Number of Peaks : number of fitting peaks

Options of background estimation:

  • Number of Iteration : number of iterations
  • Background Estimation : background estimation method

Options for efficiency fitting:

  • Energy : energy values
  • Efficiency : efficiency values corresponding to energies
